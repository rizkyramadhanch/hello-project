== Building

To launch your tests:
```
mvn clean test
```

To run your application (hot-reload):
```
mvn clean compile vertx:run
```

To build/package your application:
```
mvn clean compile vertx:package
```

To run your application (jar packaged):
```
java -jar target/*.jar
```
